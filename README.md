# an-lisis_ventas_python_sql_power_bi
Este proyecto analiza las ventas de una tienda ficticia usando Python, SQL y Power BI. Con Python se limpian y transforman los datos del archivo Excel. Luego, con SQL se guardan y analizan los datos dentro de una base de datos.Finalmente, con Power BI se crean reportes interactivos para visualizar el rendimiento de ventas, clientes y productos.

En resumen, combina estas tres herramientas para convertir datos crudos en información útil para tomar mejores decisiones.
